Helix 是一个模式文本编辑器，内置支持多重选择、语言服务器协议（LSP）、tree-sitter，以及实验性的调试适配器协议（DAP）支持。

今天我们迎来了 22.05 版本的发布，这是一个功能丰富的版本，贡献者数量是上一版本的两倍多（110 位！）。非常感谢所有参与的人！🙌

同时也要特别感谢 [`@amirrezaask`](https://github.com/amirrezaask) 制作的这个精彩的 [Helix 介绍视频](https://www.youtube.com/watch?v=__U4A2Q9fNI)！

让我们来看看 22.05 的炫酷功能。

## 缩进系统重写

缩进系统已被完全重写。缩进仍然通过查询文档的 tree-sitter 语法树来工作，但现在查询文件可以使用 tree-sitter 查询的全部表达能力，而不仅仅是简单的节点名称。期待更健壮的缩进和更少的边界情况。

## 可配置的边栏

边栏现在可以在您的 `config.toml` 文件中进行配置。例如，您可以在仍然显示 LSP 诊断信息的同时移除行号边栏。

```toml
[editor]
gutters = ["diagnostics"] # 默认是 ["diagnostics", "line-numbers"]
```

## 本地语言配置

语言配置现在可以在项目本地进行指定。创建一个 `.helix` 目录并在其中放置 `languages.toml` 文件，以覆盖[默认语言配置]中的值。例如，覆盖 C 项目的缩进设置，使用制表符而不是空格进行缩进：

```toml
# my_c_project/.helix/languages.toml
[[language]]
name = "c"
indent = { tab-width = 8, unit = "\t" }
```

## 标尺

{{ asciinema(id="rulers", width="94", height="25") }}

垂直标尺现在可以通过 `rulers` 选项进行配置。在遵守最大行长度时，将它们作为参考线使用。

## 显示可见空白字符

{{ asciinema(id="visible-whitespace", width="94", height="25") }}

空格、制表符和换行符等空白字符现在可以渲染显示。使用 `whitespace.render` 选项，可以在运行时通过 `:set` 命令或在 `config.toml` 的 editor 部分进行设置。可见的空白字符指示器可以让 Helix 的选择模型更加直观，尤其是在行尾附近。用于制表符、空格、不间断空格和换行符的字符也可以自定义，以及是否渲染这些字符。实际上，在本文剩余的部分中，您将看到我自定义的制表符和换行符！

## 在提示符中使用寄存器

{{ asciinema(id="insert-registers-in-prompts", width="94", height="25") }}

存储在寄存器中的值现在可以在搜索或全局搜索等提示符中使用。在上面的示例中，我们使用 `y` 将选中的内容拖拽到 `"` 寄存器，然后使用 `Space` `/` 打开全局搜索提示符。可用的寄存器显示在提示符上方，我们按下 `"` 来插入 `"` 寄存器中的内容。

## 重新排版段落

{{ asciinema(id="reflow", width="94", height="25") }}

新的 `:reflow` 命令将文本硬换行到指定的列宽。默认列宽可以按语言进行配置。Reflow 会识别行之间匹配的前缀，因此您可以重新排版行注释、Markdown 引用或带有前导空白的行，并且前导字符会被适当地放置。

## 交换窗口

{{ asciinema(id="window-swapping", width="94", height="25") }}

窗口现在可以与相邻窗口交换，分割也可以转置。在窗口菜单打开时（`C-w` 或 `space w`），使用 `H`、`J`、`K` 和 `L` 分别将当前窗口与左、下、上、右相邻窗口交换。使用 `t` 或 `C-t` 在垂直分割和水平分割之间切换。

## 在运行时设置语言

{{ asciinema(id="set-language", width="94", height="25") }}

缓冲区的语言现在可以在运行时进行设置。使用 `:set-language language`（或简写为 `:lang language`）来设置语言并切换语言服务器。

## 总结

这些只是 22.05 版本变化中的一些亮点。查看完整的[更新日志]了解全部内容。在[发布页面]获取 22.05 的发布二进制文件。从这个版本开始，发布二进制文件中包含了 AppImage，方便在 Linux 上使用。

期待 7 月份即将发布的 Helix 22.07 中更多的功能和改进。
